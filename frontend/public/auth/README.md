# Auth illustrations

Corner illustrations for the signup/login screens, exported from the Figma file
`BZH4CuPAaC0S7hKoItTInX` (Notes Taking App Challenge):

| File               | Figma node | Shown on                       |
| ------------------ | ---------- | ------------------------------ |
| `signup-cats.png`  | 34:899     | `/signup` ("Yay, New Friend!") |
| `login-cactus.png` | 143:244    | `/login` ("Yay, You're Back!") |

The password "peek" toggle (Figma glyph 143:247) is rendered as an inline SVG in
`src/shared/ui/password-input`, so it needs no asset here.

## Re-exporting

These are raster exports. To refresh them from Figma, re-export the nodes above
at 2Ă— as PNG and save them with the filenames in the table.
